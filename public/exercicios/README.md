# Pasta de exercícios (PDFs)

Coloque aqui os arquivos PDF de exercícios. Cada arquivo precisa ser
registrado em `src/data/exercises.ts` para aparecer no site — só
colocar o PDF aqui não é suficiente.

Exemplo de nome de arquivo: `celula-questoes.pdf`

Veja `src/data/exercises.ts` para o formato de registro completo.
